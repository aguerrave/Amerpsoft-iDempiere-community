HOW To Install Export Packages on a Development Database
ZIP Type
1. Create a generic Pack-in record
2. Attach Export Package file
3. Import the Export_XXXXXX Package
4. On Pack-out Window you will have the Records
CSV Type
1. Create a Pack-out Record
2. Set Pack-out Name equal to original Pack-out as in column 1
3. On Detail Window import CSV File
